# Module 3: The AI-Robot Brain (NVIDIA Isaac)

## Overview
This module introduces **Advanced Perception and Training** using the NVIDIA Isaac ecosystem.

## Learning Objectives
- Use **NVIDIA Isaac Sim** for photorealistic simulation and synthetic data.
- Implement **Isaac ROS** for Hardware-accelerated VSLAM.
- Use **Nav2** for path planning in bipedal robots.

## Key Concepts
### 1. Isaac Sim
A simulation platform enabling photorealistic virtual environments for AI training.

### 2. Visual SLAM (VSLAM)
Using vision (cameras) to Map an environment and Localize the robot within it simultaneously.

### 3. Nav2
 The Navigation Stack 2, allowing your robot to move autonomously from point A to B while avoiding obstacles.
